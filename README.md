#  AI Resume Builder

AI Resume Builder is a modern web application that helps users create clean, professional resumes with ease.  
It provides an interactive interface, real-time preview, and multiple resume templates to simplify the resume-building process.

---

##  Key Features

- Create and manage resume sections such as personal details, skills, education, and experience  
- Real-time resume preview while editing  
- Multiple resume templates to choose from  
- Data persistence using local storage  
- Clean and user-friendly UI  
- Built with scalable and maintainable TypeScript architecture  

---

##  Tech Stack

- **Frontend:** React + TypeScript  
- **Styling:** Tailwind CSS  
- **State Management:** React Context / Hooks  
- **Storage:** Browser Local Storage  
- **Package Manager:** npm / Yarn  



---

## Prerequisites

Make sure you have the following installed:

- Node.js (v18 or above recommended)
- npm or Yarn
- VS Code (recommended)
